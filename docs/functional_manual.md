# Manual Funcional: SansolPos

## Introducción
Bienvenido a **SansolPos**, la nueva plataforma ágil para la gestión de su punto de venta. Este manual cubre los módulos principales para el rol de cajero/administrador.

## 1. Módulo POS (Caja)
El terminal de punto de venta ha sido diseñado para ser ultra-rápido:
- **Catálogo Visual**: En el centro de la pantalla verá los productos con sus fotos y precio de venta.
- **Stock en Tiempo Real**: Si un producto baja del mínimo, se marcará estratégicamente.
- **Carrito Dinámico**: Al hacer clic en un producto se añade al panel lateral derecho. Desde ahí podrá modificar cantidades, agregar descuentos o remover el ítem.

## 2. Proceso de Checkout Obligatorio
Una vez que el carrito está completo, presione **Cobrar**.
La pantalla de checkout le exigirá:
1.  **Forma de Pago**: Efectivo, Transferencia, QR, o Tarjeta de crédito.
2.  **Celular del Cliente**: Permite posteriormente el envío opcional del comprobante.
3.  **Confirmar Venta**: Al clickear confirma la operación y descuenta inmediatamente el stock de su inventario.
Una vez confirmada, se generará el ticket.

## 3. Emisión de Tickets y Whatsapp
El comprobante se previsualiza en la pantalla de éxito.
- Puede darle al botón **Imprimir** (formato ticketera térmica habilitado).
- Puede presionar **Enviar Comprobante por WhatsApp**. Esto abrirá la aplicación Web o App de WhatsApp de forma automática enlazando el ticket para el número documentado.

## 4. Gestión de Productos
(Solo Administradores y Managers)
- Ingrese al Dashboard > Productos.
- Presione **Importar** para subir un archivo Excel/CSV con la lista de precios. El sistema detectará duplicados automáticamente.
- La tabla de movimientos guarda registro de cada alta, baja y venta para justificar el stock existente.

## 5. Módulo Gastos
Sirve para extraer la caja chica durante el turno.
- Seleccione **Gastos** en el menú.
- Ingrese la categoría (Ej: Limpieza, Publicidad, Sueldos).
- El gasto reducirá las ganancias netas que visualizará en el panel de fin de mes de su Dashboard.
